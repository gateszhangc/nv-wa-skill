# Clay Circuit

Clay Circuit 把“造人”和“造思维工具”视为同一件事的两种尺度。画面不做神话插图，而是把泥土、脑回路、实验记录与档案标注压缩成一种冷静又带温度的视觉语言。每一个轮廓都像手工反复修整后的痕迹，既保留材料感，又保留系统感。

这个方向的核心不是装饰，而是把“提炼认知框架”变成一件看得见的事。外轮廓必须像被慢慢塑出来的器皿，内层结构则像一张被推演过无数次的推理图。外层说的是形成，内层说的是验证，两者一起把女娲.skill 的方法论藏进图形里。

颜色不走科技产品惯用的紫蓝渐变，而是使用近黑底色、陶土红、玉石青与微微发亮的金色。这样既能保留中国语境里的“泥土”和“造物”联想，也不会落入传统东方审美的古风模板。颜色的关系应该像实验室灯光照在手工材料上，而不是营销页上的糖霜。

构图要像一本被顶级研究者和顶级设计师共同完成的田野手册：有秩序，有留白，有编号，有结构线，但绝不呆板。它要看起来像花了很长时间才把复杂东西压缩到这么克制的程度。重点不是堆更多元素，而是让每一条线、每一个间距都显得经过深思熟虑。

最终成品必须同时满足两个目标：第一眼足够有辨识度，第二眼还能读出方法论的野心。它不应该像传统 Logo，也不应该像随机几何图。它应该像一个能持续生长的符号系统，像女娲自己一样，把原始材料慢慢塑成一个能工作的东西。
